# Pod Collaboration Model — KoreField Academy

## What is a Pod?

A Pod is a multidisciplinary delivery team of learners assigned to simulate a real-world project team. Pods collaborate on exercises, projects, and peer reviews within a Track.

## Pod Composition

### Standard Roles (Required)
| Role | Description |
|------|-------------|
| Product Manager | Owns delivery planning and stakeholder communication |
| Data Scientist | Handles data analysis, modelling, and insights |
| AI Engineer | Builds AI systems, pipelines, and integrations |
| Cybersecurity / AI Security | Ensures security posture and threat awareness |
| Industry Specialist | Provides domain context and business alignment |
| Assessor | Supervises pod, reviews deliverables, scores professionalism |

### Optional Roles (Added as needed)
- Data Engineer
- DevOps / Cloud specialist
- Business Analyst
- UX / Product Designer

## Pod Assignment Rules

- Learner assigned to a pod on paid Track enrollment
- Role assigned based on enrolled Track and available pod vacancies
- Pod NOT activated until an Assessor is assigned
- If pod drops below minimum required standard roles (due to dropouts), remaining members are reassigned to another active pod with matching vacancies

## Pod Deliverables

Every pod must produce:
1. Working project prototype
2. Structured documentation
3. Governance checklist
4. Sprint review outputs
5. Final presentation or capstone defense

## Evaluation Areas (Per Pod Member)

1. Technical quality
2. Collaboration quality
3. Communication
4. Professionalism
5. Governance awareness
6. Delivery discipline

## Pod Workspace

Each pod gets a dedicated collaboration workspace with:
- Messaging
- File sharing
- Video conferencing capabilities

## Assessor Responsibilities Within Pods

- Monitor pod collaboration and activity
- Review deliverables and provide feedback
- Score professionalism across 5 dimensions
- Guide conflict resolution
- Validate readiness for capstone
- Approve or withhold certification eligibility
- Can request resubmission, flag underperformance, trigger remediation
- Auto-escalation: unreviewed submissions escalated after 10 calendar days
